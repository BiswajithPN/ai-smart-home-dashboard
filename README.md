## 🏅 Recognition & Participation
**Participated and won the Runner Prize at ZYNEX '24 — National Level Technical Symposium, PSNA College of Engineering & Technology, Dindigul.**

Nexus Home is a high-end, futuristic smart home control interface designed with a focus on **Cosmic Aesthetics** and **Ambient Intelligence**. Built entirely with pure web technologies, it pushes the boundaries of what's possible with Vanilla JavaScript and Modern CSS.

[**Live Demo 🚀**](https://ai-smart-home-dashboard.vercel.app/)

---

## ✨ Premium Features

### 🌌 Elite Aesthetics
- **Glassmorphic UI**: High-fidelity translucency with background saturation and blur filters.
* **Bento Grid Architecture**: A modular, responsive layout that scales elegantly across all screen sizes.
* **Dynamic Ambience**: Real-time "Mood Scenes" that adjust the dashboard's visual tone and feedback.

### 🧠 Integrated Intelligence
- **Nexus AI Hub**: Live-processed home insights and system health monitoring.
- **Smart Connectivity**: Real-time bandwidth tracking and active device visualization.
- **AI Chatbot**: A futuristic conversational interface for system management.

### ⚡ Technical Excellence
- **Zero Frameworks**: 100% Pure HTML5, CSS3, and JavaScript.
- **Buttery Smooth**: Optimized 60fps animations using Delta-Time rendering.
- **Realistic Footage**: Integrated security camera feeds with high-res architectural backgrounds.
